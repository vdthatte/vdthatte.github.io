Open Source MP3

Daniel Lee and I decided to build a product together. The first idea, that we came up with, was to build an open source mp3 player.
Which people could use on their website for free. While we played advertisements.

To start with I used the basic HTML5 audio.
and to built the web app from that point. I then used simple jquery functions such as play,
pause and stop. And linked images, forming my basic control board.

Daniel Lee was incharge of setting up the player in a server enabling a provision for file upload.
Using PHP and mySQL.
Like many of my projects, this will always go through numerous number of improvements and revisions.
My next goal is to enable the player with a playlist generating option using javascript. Stay tuned for more information.
